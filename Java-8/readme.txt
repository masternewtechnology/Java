This project is to try out all new features in java 8

New features in Java
--------------------

1) Interface Default and Static Method
		Before Java 8, Interface could have only abstract method, also if we add a new abstract method in interface, 
		then we need to implement that method in all the class that implement the interface. To over come this restriction 
		from java 8 they introduce static and default method.
		
	Static Method
	-------------
			The static method in interface is available only through and inside of a interface. It cannot be overridden by 
			implementation class
	Default Method
	-------------
			Default methods are declared using the new default keyword. This are accessible through the instance of the 
			implementation class and can be overridden.
